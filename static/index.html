Feel the market

english Stockle is a Web app and an API that provides sentiment analysis of the stock trade market. The user follows a set of companies, and the API retrieves news, reddit comment threads and tweets from Twitter. Each item is associated with an indicator of the sentiment towards the company, shown as a box plot of its distribution. The aggregated sentiment is also shown per each source of information.

Stockle API is a wrapper around the APIs of Yahoo Finance, reddit, Twitter, and AlchemyData News. It uses Alchemy Sentiment API to extract the sentiment information from texts and URLs.

In this demo, the data is retrieved from a cache updated every hour. The final product will be shipped as a Bluemix app.

italiano Stockle è una app e una API che analizza il sentimento dei mercati finanziari. L'utente può seguire una serie di aziende, e la API recupera notizie, thread di commenti da reddit e tweet da Twitter. Ognuno di questi è associato ad un indicatore del sentimento espresso verso la relativa azienda, visualizzato come un box plot della sua distribuzione. Inoltre, viene visualizzato anche il sentimento aggregato per ogni fonte di informazione.

Stockle API è uno strato di software che incapsula le API di Yahoo Finance, reddit, Twitter, and AlchemyData News. Utilizza Alchemy Sentiment API per estrarre il sentimento da testi e URL.

In questa demo, i dati sono letti da una cache aggiornata ogni ora. Il prodotto finale sarà distribuito su Bluemix.
